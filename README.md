# Shipment Tracker

A full-stack shipment tracking application built with Laravel.

## Features

-  View list of shipments with pagination
-  Search shipments by tracking number
-  Responsive design with Tailwind CSS
-  Status timeline for each shipment
-  Server-side rendering with Blade templates

## Tech Stack

- **Framework**: Laravel 12.x
- **Database**: MySQL/PostgreSQL/SQLite
- **Frontend**: Blade Templates + Tailwind CSS
- **Package Manager**: Composer

## Prerequisites

- PHP >= 8.1
- Composer
- Node.js & NPM (for Tailwind CSS)
- MySQL/PostgreSQL/SQLite
